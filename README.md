# 2-Bit memory circuit build with D latches. 
2-bit memory circuit built using D latches on a breadboard. Stores binary data using basic sequential logic with a write-enable control. Input is set through push buttons, and the current stored value is displayed on a 16×2 LCD in real time.

The push buttons can be used to put the circuit in a "Locked" or "Editable" state, where in the Locked state the outputs remain as they are. However, in the editable state, the outputs can be changed using the DIP Switch. The outputs are then given to the arduino, which uses an LCD display to display both the outputs, and the current state of the circuit.
